01. 데이터를 분석하다
데이터는 수치(문자 포함)의 모음으로 데이터 분석에는 통계학이 근간이 되고 확률론을 사용한다. 
데이터 분석의 목적은
1) 요약: 평균 등을 사용해 데이터의 경향을 파악하는 것
2) 대상 설명: 성질과 관계성 정략적이고 객관적으로 파악하는 것. 이때 설명에는 인과관계와 상관관계라는 수준 차이가 있다
3) 예측: 얻을 데이터를 기반으로 새로운 데이터를 예측하는 것. (선형관계와 기계학습)


02. 통계학의 역할
- 통계학을 통해 데이터의 퍼짐 정도 및 산포를 설명하고 예측할 수 있다
- 퍼짐이란 데이터 하나하나 값의 차이이며, 데이터 파악을 어렵게 만드는 요소
- 통계학은 확률론을 통해 터짐에 대처한다

03. 통계학의 전체 모습
- 기술 통계: 수집한 데이터를 정리하고 요약하는 방법. 확보한 데이터에만 집중(평균, 분산 등)
- 추론 통계: 수집한 데이터(표본)로부터 데이터의 발생원(모집단)을 추정하는 방법을, 데이터의 발생원에 집중
- 확률 모형: 데이터는 확률 장치에서 생성되었다는 가정에서의 확률 장치
예) 주사위는 각 눈이 1/6로 나타나는 확률 모형

- 추론 통계는 통계적 추론과 가설검정으로 나뉜다
1) 통계적 추론
- 데이터에서 가정한 확률 모형의 성질을 추정하는 방법
2) 가설검정
- 가설을 세운 후 이것이 얻은 데이터와 들어맞는지를 평가하여, 가설 채택 여부를 판단하는 방법

- 통계분석에는 데이터 유형, 변수의 개수, 가정하는 확률 모형이 영향을 미친다
- 데이터 유형은 범주형 변수와 양적 변수가 존재
예) 혈액형 / 예) 키

04. 데이터 분석의 목적과 알고자 하는 대상

데이터 분석 단계
1) 목적 설정
예) 신약의 효과 유무와 효과의 크기를 알기
2) 알고자 하는 대상을 명확히 하는 것
예) 고혈압이 있는 ‘모든’ 사람(의 혈압), 이때 ‘모든‘이 중요한 이유는 80명을 모집해 실험을 하더라도 실험의 목적은 “80명에게 신약이 효과있는지”가 아닌 “모든 환자에게 신약이 효과있는지”이기 때문

05. 모집단
모집단: 
- 알고자 하는 대상 전체
예) 신약 사례: 대상은 고혈압 환자, 모집단은 신약 복용 자가 포함된 모집단과 위약 복용자가 포함된 모집단 2개
-> 실제로 데이터를 얻을 가능성이 없는 요소를 포함한 모집단은 적절하지 않다
예) 모든 고혈압 환자를 대상으로 하고 싶지만 모종의 이유로 여성 환자의 데이터를 얻을 수 없는 경우에는, 고혈압 남성 환자를 모집단으로 설정하는 것이 적절
=>  '지금 알고자 하는 대상은 무엇인지' 그리고 '무엇을 모집단으로 설정할 것인지' 정하는 것이 중요하다

- 모집단의 크기: 모집단에 포함된 요소의 수, 크기에 따라 유한모집단과 무한모집단으로 구분
1) 유한 모집단
- 모집단 중 한정된 요소만 포함한 것
예) A,B반 성적 대항전에서 학생 80명
2) 무한모집단
- 모집단 중 포함된 요소의 개수가 무한한 것
예) 신약 사례에서 미래 고혈압 약 복용 환자도 고려해야 하기에 요소의 개수가 무한
=> 무한 모집단은 유한모집단과 달리 모든 요소를 조사하는 것은 불가능

06. 모집단의 성질을 알다
- 모집단은 데이터 분석에서 알고자 하는 대상 전체 지칭
-> 이때 모집단의 성질을 알면 대상을 설명할 수 있고, 예측도 가능
예) 신약을 복용한 사람의 최고 혈압 평균은 120mmHg이다.

- 모집단의 성질을 조사하는 방법
1) 전수조사
- 모집단에 포함된 모든 요소를 조사하는 방법
- 유한모집단일 때만
-> 이 경우 분석할 데이터=모집단 이 된다
-> 앞선 확보한 데이터에만 집중하는 기술통계에 해당
- 비용과 시간으로 인해 실현이 현실적으로 불가능

2) 표본조사
- 현실적인 측면에서 모집단 전체를 조사하지 않고 모집단의 일부를 분석하여 모집단 전체의 성질을 추정하는 추론 통계에 해당
- 표본: 추론통계에서 조사하는 모집단의 일부
- 표본 추출: 모집단에서 표본을 뽑는 것
- 표본 조사: 표본을 이용해 모집단의 성질을 조사(추정)하는 것
=> ‘추론’통계라는 용어처럼 모집단의 성질을 100% 알 수는 없으며 어느 정도의 불확실성을 고려

표본 크기
- 정의: 표본에 포함된 유한한 요소의 개수
-> 일반적으로 알파벳 n으로 표기
-> 모집단의 성질을 추정할 때의 확실성이나 가설검정의 결과에도 영향을 미치기에 중요

- 샘플 수: 표본의 개수를 의미
예) 20명으로 구성된 표본 A와 30명으로 구성된 표본 B가 있다면 표본의 개수는 2

Q. 한 회사가 직원들의 ‘업무 만족도’를 파악하기 위해 설문조사를 실시했다. 전체 직원 1,000명 중 50명에게만 설문을 진행했고, 그 결과 평균 만족도는 4.5점(5점 만점)이었다. 회사는 이를 근거로 “우리 회사 직원들은 대체로 업무 만족도가 높다”고 결론 내리려 한다.

이 결론이 타당한지 판단하기 위해 가장 먼저 확인해야 할 것은 무엇인가?
4번, 표본조사는 모집단의 전수조사가 현실적으로 불가능할 때 사용하는 방법으로, 이때 표본은 모집단의 성질을 100% 반영하지 못하며, 어느 정도의 불확실성을 감안해야 한다. 따라서 표본 50명이 모집단인 전체직원의 성질을 얼마나 반영하고 있으며, 일반화할 수 있는지를 가장 먼저 판단해야 하기에 4번이 정답이다.
