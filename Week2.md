## 02. 데이터 분포

```
✅ 학습 목표 :
* 변수 유형에 따라 히스토그램 정의를 설명할 수 있다.
```

- 데이터 분석의 첫 단계는 데이터가 어떻게 분포되어 있는지'를 그래프 등으로 시각화하여, 대략적인 데이터 경향을 파악하는 것
데이터가 어떻게 분포하는지를 파악하는 것

- 여기에는 히스토그램을 사용
1) 이산변수의 히스토그램
- 가로축은 숫자, 세로축은 데이터에 나타난 개수(도수, 빈도, 횟수)를 표시
2) 연속변수의 히스토그램
- 그때그때 설정하는 구간폭에 따라 세로축의 막대 높이가 결정된다
3) 범주형 변수의 히스토그램
- 가로축에는 각 범주를, 세로축에는 각 범주에 속하는 개수
- 대소관계가 없어 , 가로축 순서에 의미가 없다

- 히스토그램만으로는 데이터를 정확히 기술하거나, 대상을 이해하지 못한다.
- 데이터에 대해 다양한 계산을 수행하고, 수치적으로 다루는 통계분석이 필요

## 03. 통계량
```
✅ 학습 목표 :
* 통계량의 개념과 역할을 이해한다.
* 대푯값의 종류와 각 대푯값의 특성을 설명할 수 있다.
* 자료의 산포를 나타내는 분산과 표준편차의 의미를 이해한다.
* 이상값(이상치)의 개념과 통계 분석에서의 영향을 이해한다.
```
통계량: 수집한 데이터로 이런저런 계산을 수행하여 얻은 값
기술,요약 통계량: 데이터 그 자체의 성질을 기술하고 요약하는 통계량
예) 평균값
- 주대상은 양적 변수
- 통계량으로 요약한다는 것은 데이터 일부를 버린다는 것
예) 평균값에는 데이터가 퍼져있는 정도가 포함 X

대푯값: 
- 대략적인 분포 위치와 '어디를 중심으로 분포하는지를 알 수 있다
- 즉 대표적인 값을 정량화하기 위해 사용하는 통계량
1) 평균값
2) 중앙값: 크기 순으로 값을 정렬했을 때 한가운데 위치한 값
3) 최빈값: '데이터 중 가장 자주 나타나는 값
- 분포가 좌우대칭에 가까운 봉우리 형태라면 평균값, 중앙값, 최빈값은 대체로 일치

이상값이 대푯값에 끼치는 영향:
- 평균값은 계산 시 모든 값을 고려하기 때문에 이상값의 영향을 받기 쉽다
- 중앙값은 가운데에 있는 값만 참조하므로 이상값에는 잘 영향받지 않는다

통계량과 히스토그램을 같이 사용해야 한다
- 연령분포의 평균값이 31.6세인 공원에서 히스토그램 없이 이 값만으로 데이터를 이해하려 하면, 성인이 이용하는 공원이라 생각할 수 있지만, 실제로는 어린아이와 고령자가 사용하는 공원일수도

표본분산
- 데이터가 '어느 정도 퍼져 있는지(흩어져 있는지)'를 파악하는 것
- 표본의 각 값과 표본평균이 어느 정도 떨어져 있는지를 평가하는 것
- 각 값과 평균값의 차이를 제곱한 것을 모두 더한 다음 표본크기로 나눈다
- 데이터 퍼짐 상태를 정량화한 통계량
- 늘 0 이상이며, 모든 값이 같을 때는 0
- 데이터 퍼짐 정도가 크면 값이 커진다

상자수염그림
- 제1사분위수: 큰 쪽부터 세었을 때 1/4 위치에 있는 값
- 제3사분위수: 작은 쪽부터 세었을 때 1/4 위치에 있는 값
- 제2사분위수: 중앙값
- 제1사분위수부터 제3사분위수까지의 범위가 바로 상자
- 제1사분위수와 제3사분위수의 차이의 1.5배 길이를 상자로부터 늘인 범위: 최댓값, 최솟값
-  이 범위에 포함되지 않은 값은 이상값

이상값
- 평균값에서 표준편차의 2배 또는 3배 이상 벗어난 숫자
- 데이터 분석 현장에서는 시각화를 통해 이상값이 있는지를 확인하고, 그것이 실제 값인지 혹은 실수로 생긴 값인지를 판단하는 것이 중요

## 04. 확률

```
✅ 학습 목표 :
* 확률의 기본 개념과 확률변수 및 확률분포의 의미를 이해한다.
* 추론통계에서 사용되는 확률분포의 개념을 이해한다.
* 사건 간 독립의 개념과 통계적 의미를 이해한다.
```

확률
- 발생 여부가 불확실한 사건의 발생 가능성을 숫자로 표현한 것
- 영문자 P로 표기, 사건 A의 확률은 P(A)
- 확률변수: A와같이 확률이 달라지는 변수
- 실현값: 확률변수가 실제로 취하는 값, 여기서는 A
- 이산형 확률변수와 연속형 확률변수로 나뉘는데, 둘의 차이는 확률 분포에서 나타난다

확률분포
- 가로축에 확률변수를, 세로축에 그 확률변수의 발생 가능성을 표시한 분포
- 이산형과 범주형 확률변수는 확률분포가 막대그래프 형식이다
- 연속형 확률변수는 확률밀도함수를 통해서 확률을 계산한다
- 확률밀도함수의 y값이 아닌 범위의 면적이 확률이다

확률분포가 중요한 이유
- 추론통계의 목적은 모집단의 일부인 표본을 통해 모집단의 성질을 추정하는 것
- 확률분포의 의의는 '모집단과 표본 데이터'처럼 다루기 어려운 대상이 '확률분포와 그 실현값'처럼 다룰 수 있는 대상으로 치환하는 것

기댓값: 
- '변수가 확률적으로 얼마나 발생하기 쉬운가‘를 평균적으로 나타낸 것=평균
-  E(X)로 표기

분산:
- 확률분포가 기댓값 주변에 퍼져진 정도를 나타내는 값
- V(X)로 표기
- 표준편차는 분산의 제곱근

왜도
- 분포가 좌우대칭에서 어느 정도 벗어났는지 확인하는 값

첨도
- 분포가 얼마나 뾰족한지와 그래프의
꼬리가 차지하는 비율(분포의 양쪽 끝 꼬리의 확률 크기)이 얼마인지

동시확률분포
- 확률변수 2개를 동시에 생각할 때의 확률분포
- 독립: 한쪽이 어떤 값을 취하든지, 다른 한쪽의 발생 확률은 변하지 않는다는 것
- 이때 P(X,Y) = P(X)×P(Y)

조건부확률
- 한쪽 확률변수 Y의 정보가 주어졌을 때, 다른 한쪽 확률변수 X의 확률
-  P(XㅣY)로 표기, 오른쪽에 조건을, 왼쪽에 확률변수
- X와 Y가 독립인 경우에는 P(XY)=P(X)가 성립
  
## 05. 이론적인 확률분포

```
✅ 학습 목표 :
* 모수의 개념과 통계적 의미를 이해한다.
* 정규분포의 특성과 표준화의 개념을 설명할 수 있다.
```

모수(파라미터)
- 이론적인 확률분포는 수식으로 표현되며, 분포의 형태를 정하는 숫자

정규분포
- 연속형 확률변수를 대상으로 정의
- 확률분포가 평균과 표준편차라는 2개의 모수로 정해진다
- 평균=0, 표준편차=1인 정규분포 N(0,1)를 표준정규분포라고 부른다
- 평균은 분포의 위치를, 표준편차는 분포의 넓이를 결정한다
 특징
- 평균을 중심으로 한 종형으로, 좌우대칭 분포이다
- 평균 근처에 값이 가장 많고, 평균에서 멀어질수록 적어진다
- 키나 몸무게 등 정규분포로 근사할 수 있는 현상이 많다

표준화:
- 데이터의 평균과 표준편차를 이용하며 평균 , 표준편차 1로 변환할 수 있는데 이 작업이 표준화
- z값이며, z-score
- 모든 데이터를 동일한 기준으로 비교할 수 있게 만드는 것
- 내 점수가 평균보다 얼마나 잘했는지를 나타내는 상대적인 값을 구할 수 있다
- 평균과의 거리가 표준편차의 몇 배인가를 나타낸다
예) 성인 남성의 키 분포가 평균 167.6, 표준편차 7.0인 정규분포에서 180cm인 사람은 평균화를 하면 (180-167.6)/7=1.7이 된다

## 06. 추론통계를 배우기 전에

```
✅ 학습 목표 :
* '데이터를 얻는다는 것'의 의미를 이해한다.
* 무작위추출의 개념과 통계적 필요성을 설명할 수 있다.
* 추론통계의 직관적 의미를 이해한다
```

데이터를 얻는다는 것
- 모집단에 포함된 전체 값으로 구성된 분포에서 일부를 추출하는 것
- 확률분포가 결정되면, 그 확률분포를 따르는 실현값이 발생한다
예) 시험 점수 분포가 있다고 할 때
학생 A: 82점
학생 B: 65점
학생 C: 74점
이런 실제로 나온 점수가 바로 실현값
- 실현값은  데이터와 비슷하다
- 그리고 확률분포와 실현값의 관계는 모집단과 표본의 관계와 매우 비슷하다
- '모집단=확률분포','표본=확률분포를 따르는 실현값‘
- 즉, 얻은 실현값으로 이 값을 발생시킨 확률분포를 추정한다는 것의 추론통계의 핵심
- 이때 모집단분포의 모습을 들쑥날쑥한데, 이를 수학적으로 다루기 용이하게 하기 위해서 수식을 사용하고, 확률분포에 빗대어 분석을 진행하는데
- 수학적인 확률분포로 모집단분포를 근사하는 것을 모형화라고 부른다

무작위 추출
- 모집단에서 표본을 얻을 때, 모집단에 포함된 요소를 하나씩 무작위로 선택하여 추출하는 방식
- 단순무작위추출법: 표본에 있을 수 있는 모든 요소를 목록으로 만들고, 난수를 이용하여 표본을 정하는 것
- 층화추출법: 모집단을 몇 개의 층(집단)으로 미리 나눈 뒤, 각 층에서 필요한 수의 조사대상을 무작위로 추출하는 방법
- 모집단 설정에도 유의
예) 자신의 소속 대학 학생만을 대상으로 데이터를 얻어 분석했다고 하면, 그 결과가 대학생 전체를 반영한다고 말하기는 어렵다. 이는 각 대학교가 학력 수준이 다른 집단일 수 있기 때문. 그러나 건강 상태 같은 데이터라면 일부 대학교의 데이터라도 대학생 전체로 일반화할 수 있다

## 07. 표본오차와 신뢰구간

```
✅ 학습 목표 :
* 표본오차의 개념을 이해한다.
* 중심극한정리의 기본 개념과 역할을 이해한다.
* 신뢰구간의 의미와 활용 방법을 이해한다.
```

표본오차
- 표본평균을 통해서 모집단평균을 알고자 할 때 두 값은 보통 일치하지 않는다
- 이때 알고 싶은 모집단 데이터와 가지고 있는 실제 데이터 사이의 괴리를 표본오차라고 한다
- 피할 수 없음
- 표본은 모집단의 성질과 정확히 일치하지 않고, 확률오차를 수반한다

큰 수의 법칙
- 표본크기 n이 커질수록 표본평균가 모집단평균에 한없이 가까워진다는 법칙
- 표본오차 역시 0에 한없이 가까워진다
- 표본오차의 확률분포를 알면 어느 정도 크기의 오차가, 어느 정도의 확률로 나타나는지를 알 수 있게 된다

중심극한정리
- 모집단이 어떤 분포이든 간에, 표본크기이 커질수록 표본평균의 분포는 정규분포로 근사할 수 있다는 것을 의미
- 이를 통해 표본평균의 분포의 2가지 파라미터인 평균과 표준편차를 알 수 있다

추정량
- 모집단의 성질을 추정하는 데 사용하는 통계량
- 추정량은 확률변수
- 일치추정량: 표본크기 n을 무한대로 했을 때, 모집단의 성질과 일치하는 추정량
- 비편향추정량: 추정량의 평균값(기댓값)이 모집단의 성질과 일치할 때의 추정량, 모집단의 성질을 과대하지도 과소하지도 않게 나타내는 양
- 표본평균은 평균적으로 모집단 평균을 정확하게 기 때문에 비편향추정량이다
