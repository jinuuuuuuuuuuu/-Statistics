# 1️⃣ 개념 정리 
## 01. 인과와 상관

```
✅ 학습 목표 :
* 인과관계와 상관관계의 차이점에 대해 설명할 수 있다.
* 인과관계와 상관관계의 다양한 사례를 읽고 이해한다.
```
<!-- 새롭게 배운 내용을 자유롭게 정리해주세요.-->
인과관계를 이해하는 것은 곧 세상의 구조를 이해하는 것

인과관계:
원인과 결과의 관계

상관관계:
데이터에서 보이는 관련성

예) 아침밥(독립)과 성적(종속)의 인과관계 연구에 가정환경이라는 외부 변수 존재
-> 두 변수에 관련된 외부 변수가 존재할 때, 이를 중첩요인이라 부른다

무작위 통제 실험
-> 독립 변수를 무작위로 할당했을 때 중첩 요인을 제거해 인과관계를 파악할 수 있다.

허위상관: 인과는 없지만 상관은 있는 경우

인과가 무조건 상관을 의미하지도 않는다

인과관계가 중요한 이유
-> 인과가 있음으 알면 개입이 가능
- 즉 원인변수를 변화시킴으로써, 결과 변수를 바꿀 수 있다
- 개입효과를 추정 가능

상관관계도 중요한 이유
- 한 쪽 변수로 나머지 한 쪽 예측 가능
- 아이스크림 판매량과 익사율

수많은 변수를 아무렇게 해석하면, 통계적으로 유의미하다고 나타날 수 있다

## 02. 무작위 통제 실험

```
✅ 학습 목표 :
* 무작위 통제 실험에 대해 설명할 수 있다.
```

<!-- 새롭게 배운 내용을 자유롭게 정리해주세요.-->
인과관계 발견이 어려운 이유 중 하나는 중첩요인의 존재
-> 두 변수에 관련된 외부 변수가 존재할 때, 이를 중첩요인이라 부른다

예) 음주 집단과 폐암 발병률
음주가 폐암에 미치는 효과를 알고자 한다면, 음주 이외의 요인을 동일하게 하지 않으면 안된다.
-> 설명 변수에 흡연을 추가

무작위 통제 실험
- 알고자 하는 요인인 변수 X에 표본을 무작위로 할당하고, 개입 실험을 수행한 다음, 변수 Y와 비교하는 방법
예) 신약 실험군 대조군 비교 테스트
- 중첩요인을 확인하지 않더라도, 그 효과를 무작위를 이용하여 무효화해, 알고자 하는 변수의 효과만 추정 가능
- 무작위가 아니면 선택편향이 작용해 인과를 밝혀내기 어려워짐
- 즉, 중첩요인이 선택편향을 발생시켜 인과관계 파악 어려움

## 03. 통계적 인과 추론
```
✅ 학습 목표 :
* 통계적 인과 추론을 이해한다.
* 다중회귀, 층별 해석, 경향 점수 짝짓기, 이중차분법 등의 방법을 이해한다.
```
<!-- 새롭게 배운 내용을 자유롭게 정리해주세요.-->
1) 다중회귀
- 원인변수를 설명변수 x, 결과 변수를 반응변수 y로 하고, 여기에 중첩요인인 z를 설명변수로 추가한다
- 다른 변수들을 ‘통제’하고 효과만 뽑기
- 여러 조건을 고정한 상태에서 다이어트 변수의 영향만 보는 것

2) 층별해석
- 비슷한 사람끼리 그룹 나눠서 비교
- 완전히 같은 조건은 아니지만, 최대한 비슷하게 만들어 비교

3) 경향점수 짝짓기
- 다이어트 할 확률(경향점수) 계산
- 비슷한 확률 가진 사람끼리 짝짓기
- 그 안에서 비교
- > 다이어트할 가능성이 비슷한 사람끼리 비교하면 공정하다

4) 이중차분법
- 시간 변화까지 이용해서 인과효과 추정
- 원래 증가/감소 추세를 빼고, 순수 효과만 보자
- 두 집단이 원래 같은 추세였다는 가정

```
✅ 학습 목표 :

```

<!-- 새롭게 배운 내용을 자유롭게 정리해주세요.-->
통계모형

- 통계적 추론: 데이터로 모집단의 실제 분포를 추론하는 것
- 데이터를 이용해 추정한 통계모형이 실제 모집단의 분포와 어느정도 들어맞는지를 정량화함으로써, 통계 모형의 적합도를 평가할 수 있다.

  

## 05. 베이즈 통계 알고리즘

```
✅ 학습 목표 :
* MCMC 방법을 이해하고 그 사례를 설명할 수 있다.
```

<!-- 새롭게 배운 내용을 자유롭게 정리해주세요.-->
- MCMC 방법 자체는 특정 확률분포를 따르는 난수 발생 알고리즘
-  사후분포를 따르는 난수를 발생시키고, 그 난수의 집합을 관찰함으로써 사후분포의 성질을 분석

1) 몬테카를로 방법
- 난수를 여러 개 발생시켜 시뮬레이션해 근사해를 얻는 방법

2) 마르코프 연쇄
- 어떤 상태에서 다른 상태로 변화하는 현상을 확률로 표현한 모형의 일종

MCMC 방법은 그 이름대로 마르코프 연쇄와 몬테카를로 방법 두 가지를 조합하여, 사후분포를 따르는 값을 표집

## 06. 베이즈 통계 사례

```
✅ 학습 목표 :
* 베이즈 통계 사례를 읽고 이해한다.
```

<!-- 새롭게 배운 내용을 자유롭게 정리해주세요.-->
- 단계를 거듭할수록 특정 값 주변에서만 움직이고 있음을 알 수 있다
-  분포를 요약하기 위한 점 추정값으로, 평균값인 사후기댓값이나 최빈값인 사후최빈값을 계산
- 폭을 이용한 추정을 통해(1-a)% 신뢰구간 또는 확신구간을 얻을 수도 있다
- 추정 결과 95% 확률로 통계 모형의 파라미터는 이 범위에 있음을 나타내는 것

## 문제 1.

> **🧚Q. 한 연구자가 건강식품의 효과를 검증하기 위해 연구를 진행하였다.
관찰 데이터에서는 “섭취한 사람이 피로도가 더 낮다”는 결과가 나왔다.
그러나 해당 연구는 참가자가 자발적으로 제품을 구매했기 때문에, 건강 관심도나 생활습관과 같은 요인이 함께 작용했을 가능성이 있다.
이에 연구자는 참가자를 무작위로 ‘섭취 집단’과 ‘비섭취 집단’에 배정하는 무작위 통제 실험(RCT)을 설계하려 한다.
또한 기존 연구를 통해 ‘이 건강식품이 효과가 있을 확률은 60%’라는 사전 믿음을 가지고 있으며, 새로운 실험 결과를 반영해 이 믿음을 갱신하려 한다.**

> **위 상황을 바탕으로 다음을 설명하시오.**

~~~
1️⃣ 관찰 연구 결과만으로 인과관계를 단정할 수 없는 이유를 ‘상관과 인과’의 관점에서 설명하시오.
2️⃣ 무작위 통제 실험이 인과 추론에 유리한 이유를 설명하시오.
~~~

<!--상관관계와 인과관계를 구분하는 핵심 기준이 무엇인지 떠올려보세요.-->

```
1. 제품 섭취와 피도로 사이에 건강 관심도나 생활습관 같은 중첩요인이 존재하기 때문에, 섲취 이외의 요인을 동일하게 하지 않으면 상관만 알 수 있을 뿐, 인과를 모른다.
2. 건강 관심도나 생활습관 같은 중첩요인의 영향을 배제하고, 무작위로 하면 비슷단 두 집단이 형성되기 때문에 효과 검증에 유리하다.
```
