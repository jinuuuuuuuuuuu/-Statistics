# 1️⃣ 개념 정리 
## 01. 양적 변수 사이의 관계를 밝히다

```
✅ 학습 목표 :
* 상관과 회귀에 대해 이해한다.
```
<!-- 새롭게 배운 내용을 자유롭게 정리해주세요.-->
산점도: 두 개의 데이터를 2차원 평면에 나타낸 것
상관: 이때 두 확률변수의 관계성
- 단, 상관이 인과를 뜻하지는 않는다, 하나의 가능성일 뿐

회귀: 두 개의 양적 변수의 관계를 분석해 함수로 공식화하는 것
- 이때 y=f(x)에 대해서 x가 설명변수, y가 반응변수

## 02. 상관관계

```
✅ 학습 목표 :
* 피어슨 상관계수의 개념과 특징에 대해 설명할 수 있다.
* 비모수 상관계수에 대해 설명할 수 있다.
* 상관계수의 유의성 검정과 p값·표본크기의 영향을 이해한다.
```

<!-- 새롭게 배운 내용을 자유롭게 정리해주세요.-->
피어슨 상관계수 r: 상관관계에서 두 변수의 관계의 강도를 정량화한 것
- 이때 r은 -1<=r<=1 범위이다
- 양의 상관: r이 양수일 때는 변수 x, y가 정비례
- 음의 상관: r이 음수일 때는 변수 x, y가 반비례
- r의 절댓값이 클수록 상관정도가 크다

주의할 점
- 피어슨 상관계수는 ‘선형’ 관계에서만 적용
- 직선의 기울기와는 관계가 없다
- 이상값에 큰 영향을 받는다

정규성 검사와 비모수 상관계수
- x, y중 하나 이상에 정규성이 없을 때는 스피어만 순위상관계수p 권장
- 순위로 계산하므로 정규성이 없고 이상값이 있어도 괜찮다
- 비슷한 값으로 켄달 순위상관계수는 표본크기가 작을 때
- 상관관계를 계산할 때는 두 변수의 처음 관계가 종속이 아닌지 확인

상관계수와 가설
- 모집단의 상관관계가 0이라고 해도, 그로부터 추출한 표본은 0이 아닐 수 있다
- 상관이 있는지 없는지는 r이 아니라 p-value로 판단한다

표본크기와 가설검정
- 표본이 크면, 의미 없는 작은 상관도 유의하게 나올 수 있다
- p<0.05라고 해서 곧바로 상관이 있다고 판단하지 말기

비선형상관
- 피어슨 상관계수가 r=1일 때, x의 값을 알면 y의 값을 알 수 있듯이, 정보에 기반해 생각해야 한다.

## 03. 선형회귀
```
✅ 학습 목표 :
* 최소제곱법의 원리를 이해한다. 
* 회귀계수의 가설검정에 대해 설명할 수 있다.
* 결정계수에 대해 설명할 수 있다.
* 오차의 등분산성과 정규성에 대해 설명할 수 있다.
* 설명변수와 반응변수에 대해 설명할 수 있다.
```
<!-- 새롭게 배운 내용을 자유롭게 정리해주세요.-->
회귀분석이란?
- 함수 f(x)처럼 회귀식을 얻을 수 있으면 변수 사이의 관계성을 알 수 있다
- 설명변수가 하나이면 단순회귀, 여러개면 다중회귀
- y=a+bx라는 회귀식에서 식의 형태를 결정하는 파라미터 a,b가 회귀계수이다
- 평가기준에 따라 회귀의 적합도(좋음 정도)를 평가하고, 이 회귀계수의 값을 구체적으로 구해야 한다
- 이 회귀식을 사용해서 회귀를 하는 것이 회귀모형
- 회귀는 모든 점을 맞추는 게 아니라, 전체적으로 가장 덜 틀리는 선을 찾는 것

최소 제곱법
- 회귀직선을 그릴 때, 이것이 모든 데이터를 통과할 수는 없는데
- 이것이 확률오차
- 최량선형비편향추정량: 최소제곱법으로 얻은 추정량은 비편향추정량 중에서도 가장 정밀도가 높은 비편향추정량이 된다

결정계수
- 회귀식이 잘 들어맞는지 평가하는 지표
- 이 결정계수가 1에 가까울수록 회귀모형이 데이터에 잘 들어맞음을, 0에 가까울수록 잘 들어맞지 않음을 의미

설명변수와 반응변수
- 분석을 하기 전에 무엇을 설명변수로 하고 무엇을 반응변수로 할 것 인가를 목적에 맞게 생각
1) 한쪽 변수로 다른 한쪽 변수를 설명하고자 할 때
2) 인과효과를 알고 싶을 때
3) 데이터를 예측하고 싶을 때

## 04. 선형회귀 원리의 확장

```
✅ 학습 목표 :
* 다중회귀를 이해하고 그 결과를 읽을 수 있다.
* 편회귀계수의 의미를 이해하고 해석할 수 있다.
* 범주형 변수를 더미변수로 변환하는 방법과 회귀계수의 해석을 이해한다.
* 공분산분석에 대해 설명할 수 있다.
* 다중공선성에 대해 이해한다.
```

<!-- 새롭게 배운 내용을 자유롭게 정리해주세요.-->
선형회귀는 다양한 해석 방법의 기초
- 실제 데이터 해석에서는 설명 변수가 여러 개이거나 반응변수가 양적 변수가 아닌 경우도 있다.

다중 회귀
- 설명변수가 여러 개인 것
- 편회귀계수:기울기 b, b₂를 다중회귀에서 부르는 명칭
- 표준화편회귀계수: 회귀분석을 시행하기 전에 각각의 설명변수를 평균 0, 표준편차 1로 변환한 다음, 회귀분석을 시행하여 구한 회귀계수
- 상관계수가 1에 가까운 강한 상관이 있을 때는, 뒤에 설명할 다중공선성이 있는지를 의심하고 이에 대처해야

공분산 분석
- 다른 변수 영향 빼고, 그룹 차이만 공정하게 비교한 것
사용조건
1)　집단 간 회귀의 기울기가 서로 다르지 않을 것
2)  회귀계수가 0이 아니어야 할 것

다중공선성
- 설명변수가 여러 개인 다중회귀에서 설명변수 사이에 강한 상관이 있는 경우
- 분산팽창인수 VIF로 측정
